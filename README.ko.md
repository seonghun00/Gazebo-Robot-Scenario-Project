# ROS2-Gazebo-Robot-Scenarios

#### [English](README.md) | Korean

> 물류로봇과 서빙로봇의 작업 시나리오를 설계하고,  
Gazebo 환경에서 구현 과정을 기록하는 프로젝트 저장소입니다.

## 🎯 목적

물류창고와 레스토랑 환경에서 로봇이 수행할 수 있는 작업 흐름을 시나리오로 정리하고, Gazebo 기반 시뮬레이션 환경에서 단계적으로 구현하는 것을 목표로 합니다.

기본 주행부터 장애물 회피, 예외 상황 처리, 다중 로봇 협업 구조까지 확장할 예정입니다.

## ✅ 체크리스트

### 공통

- [ ] ROS2 Humble 개발 환경 구성
- [ ] Gazebo 시뮬레이션 환경 구성
- [ ] 기본 로봇 URDF 모델 제작
- [ ] 2D LiDAR 센서 추가
- [ ] SLAM 기반 지도 생성
- [ ] Nav2 기반 Waypoint 이동 구현

### 물류로봇

- [ ] 물류창고 Gazebo World 제작
- [ ] 보관구역, 포장구역, 대기구역, 충전구역 배치
- [ ] 랙 모델 제작
- [ ] 리프트형 로봇 구조 모델링
- [ ] 랙 하부 진입 및 리프트 동작 구현
- [ ] 목적지 점유 예외 상황 처리
- [ ] 교차로 통행 우선순위 제어

### 서빙로봇

- [ ] 레스토랑 Gazebo World 제작
- [ ] 주방, 픽업대, 테이블, 대기구역, 충전구역 배치
- [ ] 서빙로봇 URDF 모델 제작
- [ ] 픽업대 → 테이블 → 복귀 이동 구현
- [ ] 통로 장애물 회피 시나리오 구현
- [ ] 음식 수령 지연 상황 처리
- [ ] 충격 감지 및 비상 정지 시나리오 구현
